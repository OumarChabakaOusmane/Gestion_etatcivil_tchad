# Références Bibliographiques et Webographiques

## 1. Références Bibliographiques

*   **Pascal Roques**, *UML 2 par la pratique : Études de cas et exercices corrigés*, Éditions Eyrolles, 7ème édition, 2011.
*   **Joseph Gabay, David Gabay**, *UML 8 : De l'analyse à la conception*, Éditions Dunod, 2008.
*   **Robert C. Martin**, *Clean Code: A Handbook of Agile Software Craftsmanship*, Prentice Hall, 2008.
*   **Douglas Crockford**, *JavaScript: The Good Parts*, O'Reilly Media, 2008.

## 2. Références Webographiques (Sites Web)

### 2.1. Documentation Officielle des Technologies
*   **React.js :** [https://react.dev/](https://react.dev/) - Documentation officielle pour le développement frontend web.
*   **Node.js :** [https://nodejs.org/en/docs/](https://nodejs.org/en/docs/) - Documentation officielle de l'environnement d'exécution backend.
*   **Firebase :** [https://firebase.google.com/docs](https://firebase.google.com/docs) - Guides et références pour Firestore, Auth et Storage.
*   **Express.js :** [https://expressjs.com/](https://expressjs.com/) - Documentation du framework backend utilisé.
*   **React Native / Expo :** [https://docs.expo.dev/](https://docs.expo.dev/) - Documentation pour le développement d'applications mobiles.

### 2.2. Outils et Ressources de Design
*   **Bootstrap 5 :** [https://getbootstrap.com/docs/5.0/](https://getbootstrap.com/docs/5.0/) - Framework CSS pour les interfaces responsives.
*   **Lucide Icons :** [https://lucide.dev/](https://lucide.dev/) - Bibliothèque d'icônes utilisée dans le projet.
*   **Google Fonts :** [https://fonts.google.com/](https://fonts.google.com/) - Typographies utilisées pour l'interface.

### 2.3. Sécurité et Standards
*   **JWT.io :** [https://jwt.io/](https://jwt.io/) - Informations sur le standard JSON Web Token pour l'authentification.
*   **MDN Web Docs :** [https://developer.mozilla.org/](https://developer.mozilla.org/) - Référence complète pour le JavaScript, le HTML5 et le CSS3.
*   **OWASP :** [https://owasp.org/](https://owasp.org/) - Guide des bonnes pratiques en matière de sécurité des applications web.
